Quick Start

Generate Test Data
curl -X POST http://localhost:3001/api/solvency/simulate -H "Content-Type: application/json" -d '{"accountCount": 100}'
Get Latest Proof
curl http://localhost:3001/api/proof/latest
Verify User Inclusion
curl -X POST http://localhost:3001/api/inclusion-proof/verify -H "Content-Type: application/json" -d '{"userId": "user_0", "signature": "demo"}'
Check System Health
curl http://localhost:3001/api/health
